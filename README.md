# Ghastly_animation

## 📖 Sobre o Projeto

Este projeto consiste em uma animação visual baseada no design do GameBoy clássico, construída totalmente no navegador.

O foco principal foi praticar estruturação de interfaces, animações em CSS e organização de código Front-End.

Não se trata de um jogo interativo, mas sim de uma experiência visual animada.

---

## 🚀 Funcionalidades

- Interface inspirada no console GameBoy
- Animações visuais em CSS
- Layout responsivo
- Estrutura organizada em HTML, CSS e JavaScript

---

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript

---

## 🎯 Objetivo

Demonstrar habilidades em desenvolvimento Front-End, animações web e criação de interfaces com identidade visual temática.

---

## 📌 Observação

Projeto desenvolvido para fins educacionais e composição de portfólio.
